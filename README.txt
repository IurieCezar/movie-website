README file for the program ‘movie website’ written by Iurie Popovici.

Files included:
1. entertainment_center.py - stores the instances of the class Movie. It also calls the 'open_movie_page()' function from 'fresh_tomatoes module', this way running the whole program
2. fresh_tomatoes.py - contains the styles and scripting for the page, generates the HTML page.
3. media.py - contains the ‘class Movie’ code

Running the program
To successfully run the program the machine needs to have Python 2.7 installed. Next, the user has to navigate, using the terminal, to the folder containing the files listed above and type the following command in the terminal:

> python entertainment_center.py
